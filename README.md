# Sentinel2Py Downloader & Plotter

A Python package for downloading, stacking, and visualizing **Sentinel-2 L2A** imagery from **Microsoft Planetary Computer**.

## Features

- Search Sentinel-2 imagery by bounding box, date, and cloud cover  
- Select the least cloudy or specific tile(s)  
- Download bands with presets (RGB, NIR, SWIR, etc.)  
- Stack bands at original, highest, or user-defined resolution  
- Plot RGB composites with enhanced visualization options  
- Compute and plot
